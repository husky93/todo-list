# todo-list-app

## About
Web application for creating todo lists written in JavaScript.
<br>

#### Features:
- Store data in local storage
- Dynamically render content
- Create new projects
- Switch between projects
- Create new Todos
- Mark Todo as complete
- Show Todo details
- Delete Todo
- Edit Todo
- Show number of tasks next to the project
